# The-Gaming-Incentive
The gaming social media for gamer by gamers

## Basic Setup
Run the commands

```zsh
yarn install
```

### Android
```zsh
yarn run android
```

### IOS
```zsh
cd ios
```

For intel chip
```zsh
pod install
```

```zsh
yarn run android
```

For Apple Silicon
```zsh
arch -x86_64 pod install
```

```zsh
yarn run android
```
